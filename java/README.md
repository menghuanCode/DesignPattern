### 设计模式的类型

根据设计模式的参考书 **Design Patterns** 中提到的，总共有 23 种设计模式，
可以分为三大类：
0. 创建者模型（Create Patterns）
0. 结构型模式（Structural Patterns）
0. 行为型模式（Behavioral Patterns）

当然还会讲另一类设计模式：J2EE 设计模式。
